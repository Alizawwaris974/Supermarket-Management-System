# Supermarket Management System

## Overview
This project is a comprehensive supermarket management system developed in C++ using Object-Oriented Programming (OOP) principles. It provides a robust solution for managing supermarket operations effectively and efficiently.

## Features
- **Graphical and Command-Line Interface**: The system supports both GUI and CLI outputs, giving users flexibility in interaction.
- **Raylib Library**: Utilizes the Raylib library to deliver a visually appealing interface and enhance user experience.
- **Inventory Management**: Manages items from a `items.txt` file, allowing for easy updates and category displays to help users find products quickly.
- **Shopping Cart Functionality**: Users can add items to a shopping cart, view cart contents, and calculate total costs easily.
- **Payment Options**: Supports multiple payment methods, including:
  - **Card Payment**: Secure processing of card payments.
  - **Cash Payment**: Simple and efficient cash handling.

## Getting Started
### Prerequisites
- C++ Compiler (e.g., g++, clang++)
- Raylib library installed
  - [Raylib Installation Guide](https://github.com/raylib/raylib/blob/master/docs/README.md)

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/Alizawwaris974/Supermarket-Management-System.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Supermarket-Management-System
   ```
3. Compile the project:
   ```bash
   g++ main.cpp -o supermarket -lraylib
   ```

## Usage
Run the application using:
```bash
./supermarket
```
Follow the on-screen instructions to navigate through the system.
